# FileRename 跨平台批量更名，文件重命名工具
使用方法一：

参数形式示例：` rename  "d:\test"  "www.baidu.com","" `

在命令行输入如上命令，即可对test目录下的所有文件进行更名，将文件名包含`www.baidu.com`的关键字进行清除。
